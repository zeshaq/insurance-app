What you've built

Nine features wired across 14 slices, all clickable below. Each card is a single operation in the app. Hit the button, then flip to the matching admin UI in another tab to watch the platform service light up.

Money chain

slices 1–5

Quote

POST /api/quotes — premium calc, Redis cache + rate limit, Kafka producer, MI credit lookup, JWT-gated.

→ Kafka UI · RedisInsight · SigNoz slice 6

Policy bind

POST /api/policies — Redlock distributed lock, idempotent. Writes to log-compacted topic policy-events.

→ Kafka UI (see cleanup.policy=compact) slice 7

Payment

POST /api/payments with Idempotency-Key header. Failing amount triggers @Retry + DLQ.

→ Kafka UI (payment-events, payment-dlq)

Observability lanes

slices 9 + 10

Claim filing

Multipart upload to MinIO + OCR via MI + mTLS partner lookup. Pick a photo, optionally provide an other-party VIN.

→ MinIO console · OpenSearch (slice 14) slice 11

Agent dashboard

Live WebSocket feed of claim events + Redis Stream replay on connect. Open in one tab, file claims from another.

→ RedisInsight (dashboard:stream) slice 14

Search

Debezium CDC → OpenSearch. Search claims by description, VIN, carrier.

→ OpenSearch Dashboards slice 13

Audit trail

Compacted-topic projection vs retention-based history. Side-by-side contrast.

→ Kafka UI (audit-events, claim-events) slice 12

Reports

Kafka Streams windowed counts + a scheduled MI task that snapshots them.

→ Kafka UI · Postgres report_run table

SigNoz · Kafka UI · Mailpit · MinIO · OpenSearch · RedisInsight · WSO2 IS console · WSO2 APIM

Auth model

No login screen. On every page load the GUI calls GET /api/auth/token — server-side, Liberty mints a client_credentials JWT from WSO2 IS and hands it back. JS keeps it in memory and attaches Authorization: Bearer ... to every API call. Open DevTools → Network to see it. Real OIDC login (with users) is the next iteration; this shortcut keeps the tour focused on platform features.
